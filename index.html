Meus Projetos

Chat Online

Aplicação de chat em tempo real, onde dois ou mais usuários podem trocar mensagens instantaneamente. Utiliza tecnologias como JavaScript e WebSocket para garantir uma comunicação rápida e dinâmica. Ideal para testes de funcionalidades interativas entre usuários.

Busca de CEP

Ferramenta onde o usuário digita um CEP e recebe as informações completas de endereço, como rua, bairro, cidade e estado. Utiliza integração com API (como ViaCEP) para buscar os dados automaticamente. Um ótimo exemplo de consumo de API e exibição dinâmica de dados no front-end.

Jogo do Dino

Um jogo simples e divertido inspirado no famoso Dino do Google Chrome. Desenvolvido com HTML, CSS e JavaScript, o jogador controla um personagem que precisa desviar de obstáculos para marcar pontos. O projeto foca em lógica de colisão, controle com teclado e loop de animação.

Previsão do Tempo

Construi um app simples de previsão do tempo, onde o usuário digita o nome de uma cidade e o site exibe as informações climáticas do local, como temperatura e condição do céu. Usei integração com API externa para buscar os dados em tempo real.

Clone do Medium

Desenvolvi um clone visual do site Medium, focando no layout, responsividade e experiência de leitura. Reproduzi a interface de forma fiel, incluindo seções como lista de artigos, navegação e estilo minimalista.

Clone do ZOOM

Criei um clone da página inicial do site ZOOM, simulando a exibição de produtos, filtros e banners promocionais. O foco foi praticar design responsivo e a estruturação de componentes reutilizáveis.

Projeto Carros

Criei um site interativo de carros, onde o usuário pode clicar em cada modelo para ver a descrição completa, como detalhes técnicos e curiosidades. O projeto também conta com modo claro e escuro, oferecendo uma melhor experiência de navegação conforme a preferência do usuário.
